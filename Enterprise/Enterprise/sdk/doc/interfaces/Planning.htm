Enumeration Types

PublishPrioType

choice ofcomment
"MustHave" Advert MUST be placed
"ShouldHave" Advert should be placed (when there is enough space)
"NiceToHave" Advert can be placed (when there is splenty space)

RenditionType

choice ofcomment
"none" Request for object info only; without attachment
"preview" low res preview image (jpeg) sent as DIME attachment
"output" high res output file (PDF, EPS, etc) sent as DIME attachment (used for production)
