Language Support

Article ID: WebEdit003

Affected products: Web Editor

Affected vesions: 5

Affected platforms: All

Question

Which languages are supported by the Web Editor?

Analysis

A distinction should be made between the following areas of the application in which a language is used:

  • User interface

  • Spelling

The Web Editor version 5 uses TinyMCE version 2.1 which supports a variety of languages.

Answer

User Interface

For a full overview of the supported user interface languages, contact tinymce.

Spelling

Spelling is controlled by the installed dictionaries. The Web Editor makes use of the Google XML Service support within the TinyMCE functionality. This currently supports the following languages: English, Dutch, French, Italian, Spanish, Polish, Danish, German and Swedish.

For the latest version of the TinyMCE Spellchecker, see http://tinymce.moxiecode.com/download.php.
