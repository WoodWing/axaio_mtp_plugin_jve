Celum Server Plugin for Enterprise v6.1.x and v7

The Celum plugin implements a Content Source plugin to access content from Celum Imagine.

Configuring the Plugin

1. Open the plugin's config.php and set URL, user account and password

2. In the plugin's config.php set CELUM_BRAND, CELUM_CATEGORY, CELUM_STATUS to names of your system.

4. Enable the server plugin


Using the Celum Plugin

Open Content Station or Smart Connection and use the Celum search


Technical Note

The Celum OpenAPIPlus 4 is used for the integration.
