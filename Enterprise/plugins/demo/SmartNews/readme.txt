Smart News Integration for Enterprise 7.0.x

How to install/config

1. Install the plugin

2. Put your Smart News credentials in config.php

2. Create new publication channel and select Smart News as publishing system

3. Create a dossier, with a category, the category should also be available on the Smart News server

4. Now you are ready to publish to Smart News using the Smart News publication channel!


NOTE: To publish to Smart News, you should at least have a head element (for the title) and a body element (for the article). An intro element is optional, but preferred.
